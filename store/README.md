# 스토어 등록 에셋

Play Console 리스팅에 업로드할 이미지 모음. (모두 자동 생성 — `scratchpad/imgtool`)

| 파일 | 크기 | 용도 |
|---|---|---|
| `play-icon-512.png` | 512×512 | Play Console 고해상도 앱 아이콘 (필수) |
| `feature-graphic.png` | 1024×500 | 피처 그래픽 (필수) |
| `screenshot-1-home.png` | 1080×1920 | 스크린샷 — 홈 |
| `screenshot-2-detail.png` | 1080×1920 | 스크린샷 — 상세 |
| `screenshot-3-search.png` | 1080×1920 | 스크린샷 — 검색 |
| `screenshot-4-noti.png` | 1080×1920 | 스크린샷 — 알림 |

> 스크린샷은 최소 2장 필요, 최대 8장. 위 4장으로 충분.
> 앱 내부 아이콘(`assets/`)과 별개로, Play Console은 512×512 아이콘을 따로 요구함.

## 아직 남은 리스팅 입력 (이미지 외)
- 앱 이름 / 간단한 설명(80자) / 자세한 설명(4000자)
- 개인정보처리방침 URL (준비됨: Notion)
- 데이터 보안 양식, 콘텐츠 등급 설문, 광고 포함 여부(예)
