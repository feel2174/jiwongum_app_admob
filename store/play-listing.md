## 0. 앱 포지셔닝

**서비스 한 줄 정의**
```
50대 후반부터 70대 이상 사용자가 연금, 돌봄, 일자리, 생활비 지원 정보를 쉽게 확인하도록 돕는 시니어 혜택 안내 앱
```

**핵심 사용자**
- 은퇴 전후로 국민연금, 기초연금, 건강보험료가 궁금한 50대 후반~60대
- 노인일자리, 연금, 생활비 지원을 확인하려는 60대 이상
- 부모님 돌봄, 치매 상담, 상속 절차를 대신 확인하는 보호자

**운영 방향**
- 신청은 공식 기관 페이지에서 진행하도록 안내합니다.
- 앱은 복잡한 제도를 쉽게 이해하고 공식 확인 경로로 이동하도록 돕는 정보 서비스입니다.
- 정부 또는 공공기관의 공식 앱이 아닙니다.

## 8. 출시 트랙

개인 개발자 계정일 경우:

1. 비공개 테스트 트랙 생성
2. 테스터 12명 이상 초대
3. 14일 이상 테스트 유지
4. 프로덕션 액세스 신청
5. 승인 후 프로덕션 출시

업로드 파일:

```bash
eas build -p android --profile production
```

위 명령으로 생성된 Android AAB 파일을 Play Console에 업로드합니다.

## 9. 출시 전 최종 확인 문구

출시 전 앱 내부와 스토어 문구가 아래 내용과 일치하는지 확인합니다.

- 앱은 정부 공식 앱이 아님
- 실제 신청은 공식 홈페이지에서 진행
- 지원 대상과 금액은 기관 심사에 따라 달라질 수 있음
- 광고가 포함되어 있음
- 알림은 사용자가 허용한 경우에만 발송
- 개인정보처리방침 URL이 정상 접속됨
